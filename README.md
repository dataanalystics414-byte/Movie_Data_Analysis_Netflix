# # 🎬 Netflix Movies Analysis Using Python

## 📌 Project Overview

This project focuses on analyzing a movie dataset using Python to uncover insights related to genres, popularity, ratings, and release trends. Through data cleaning, preprocessing, and exploratory data analysis (EDA), the project transforms raw data into meaningful visualizations and business insights.

The analysis was performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing.
- Explore the structure and quality of the dataset.
- Analyze movie popularity and ratings.
- Identify the most common genres.
- Examine the relationship between genres and popularity.
- Discover trends in movie releases over time.
- Create visualizations to communicate insights effectively.

---

## 📂 Dataset Information

The dataset contains information about movies, including popularity scores, ratings, release dates, and genres.

### Dataset Features

| Column Name | Description |
|------------|-------------|
| Release_Date | Year of release |
| Title | Movie title |
| Popularity | Popularity score |
| Vote_Count | Total number of votes |
| Vote_Average | Average user rating |
| Genre | Movie genre |

## 🔄 Data Cleaning and Preprocessing

The following preprocessing steps were performed:

- Checked the dimensions of the dataset.
- Verified missing values and duplicate records.
- Converted the `Release_Date` column into datetime format.
- Extracted the release year.
- Removed unnecessary columns:
  - `Overview`
  - `Original_Language`
  - `Poster_Url`
- Handled outliers in the `Popularity` column.
- Categorized the `Vote_Average` column.
- Split and transformed the `Genre` column.

## 🧠 Key Findings

### Most Frequent Genre

Drama is the most frequent genre in the dataset, accounting for more than **14%** of all movies.

### Most Common Rating Category

More than **25.5%** of the movies belong to the **Popular** rating category.

### Most Popular Movie

**Spider-Man: No Way Home** has the highest popularity score in the dataset.

**Genres:**

- Action
- Adventure
- Science Fiction

### Least Popular Movie

**The United States, Thread** has the lowest popularity score in the dataset.

**Genres:**

- Music
- Drama
- War
- Science Fiction
- History

### Year with the Most Movies

The year **2020** contains the highest number of movies in the dataset.
